# skillswap-tiffany# Skill Swap – Sprint 1 Homepage

Skill Swap is a student-to-student platform designed to help learners discover, browse, and offer skills in a peer-to-peer environment. This repository contains the Sprint 1 homepage implementation, focusing on frontend structure, navigation, styling, and interactive skill discovery components.

The goal of this sprint is to establish professional Git workflows while building the foundation for Skill Swap’s user experience. Development follows industry-standard practices including feature branching, pull requests, and self-review before merging into the main branch.

## Sprint 1 Features
- Semantic homepage structure using HTML5
- Navigation menu with core user paths
- Styled layout for header, hero section, content area, and footer
- Interactive skill cards demonstrating basic user interaction
- Clear separation of structure, styling, and behavior

## Git Workflow
This repository demonstrates professional Git usage through:
- Feature branches for each major change
- Three merged pull requests into the main branch
- Clear, descriptive commit messages
- Inline self-review comments on pull requests

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Git & GitHub

Sprint 1 complete.
